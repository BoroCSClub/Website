## Writing Your First Pygame Script

Let’s create a simple script that opens a window using Pygame:

```python
import pygame
import sys

# Initialize Pygame
pygame.init()

# Set up the display
screen = pygame.display.set_mode((640, 480))
pygame.display.set_caption('Hello, Pygame!')

# Main game loop
running = true
while running:
    for event in pygame.event.get():
        if event.type == pygame.QUIT:
            running = false

    pygame.display.flip()

pygame.quit()
sys.exit()
```

## Running Your Code

To run your script:

1. Press the "Run" button at the top of the Replit editor.
2. You should see a new window titled "Hello, Pygame!" displaying an empty 640x480 window.
